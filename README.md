# ⛅ Weather Dashboard

A responsive, web-based historical weather dashboard that fetches and visualizes weather data using the Open-Meteo API. Built as a software engineering project for Tampere University of Applied Sciences (TAMK).

## 🚀 Features

**Real-time Data Fetching:** Retrieves hourly historical weather variables including Temperature (2m), Wind Speed (10m), and Relative Humidity (2m) directly from the Open-Meteo API.
**Data Visualization:** Interactive line charts built with Chart.js to visualize weather trends over time.
**Statistical Analysis:** Calculates and displays key mathematical statistics for the weather data, including Mean, Median, Mode, Range, Standard Deviation, and Min/Max values.
* **Reverse Geocoding:** Automatically resolves latitude and longitude coordinates into readable city and country names using the OpenStreetMap Nominatim API.
**Responsive UI:** Clean, custom-built user interface using pure HTML and CSS (no UI frameworks) with an adaptive dark/light color scheme.

## 🛠️ Technologies Used

**Frontend:** HTML5, CSS3, Vanilla JavaScript 
**Libraries:** Chart.js (for data visualization)
**APIs:** * Open-Meteo Forecast API
  * OpenStreetMap Nominatim API (Reverse Geocoding)
