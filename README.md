# ⛅ Weather Dashboard

[cite_start]A responsive, web-based historical weather dashboard that fetches and visualizes weather data using the Open-Meteo API[cite: 26]. Built as a software engineering project for Tampere University of Applied Sciences (TAMK).

## 🚀 Features

* [cite_start]**Real-time Data Fetching:** Retrieves hourly historical weather variables [cite: 67, 68] [cite_start]including Temperature (2m) [cite: 69][cite_start], Wind Speed (10m) [cite: 71][cite_start], and Relative Humidity (2m) [cite: 73] directly from the Open-Meteo API.
* [cite_start]**Data Visualization:** Interactive line charts built with Chart.js to visualize weather trends over time[cite: 44].
* [cite_start]**Statistical Analysis:** Calculates and displays key mathematical statistics for the weather data, including Mean, Median, Mode, Range, Standard Deviation, and Min/Max values[cite: 125, 127, 129, 131, 133, 135].
* **Reverse Geocoding:** Automatically resolves latitude and longitude coordinates into readable city and country names using the OpenStreetMap Nominatim API.
* [cite_start]**Responsive UI:** Clean, custom-built user interface using pure HTML and CSS (no UI frameworks) with an adaptive dark/light color scheme[cite: 43].

## 🛠️ Technologies Used

* [cite_start]**Frontend:** HTML5, CSS3, Vanilla JavaScript [cite: 43]
* [cite_start]**Libraries:** Chart.js (for data visualization) [cite: 44]
* [cite_start]**APIs:** * Open-Meteo Forecast API [cite: 30, 32]
  * OpenStreetMap Nominatim API (Reverse Geocoding)
